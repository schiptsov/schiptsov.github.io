Lotus123dos

This is an artifact form the golden age of computer tech.

This is a rare example of an evolved (via user’s feedback and demands) software system, which has been built with rational minimalism in user interface, reliability of underlying math engine, and utmost attention to details.

Some considered this being the killer app, for which users purchased a very costly PC. Yes, a whole 2-3k USD PC to run just Lotus 1-2-3 for DOS.

I don’t even know how to compare this masterpiece with modern crap.

It is important to realize that this spreadsheet software is by no means a toy or somehow inferior in any aspect. Some organizations are still using it to this day - it just works as intended.

Lets say that the full installation of 1-2-3 suite is less than fucking npm, or cargo, excluding dependencies (while everything is included in the 12MB (megabytes) of the lotus full installation).

I don’t even want to continue this writing. It makes me sick.

Just find the diskette images on some abandonware site, install and use it. This is still the killer app for so called retro-computing.

One could even buy a dedicated cheap DOS PC or a vintage laptop to run the stuff.

It literally brings back joy and awe we used to have back then in early 90s.

And yes, when one watches some crappy facebook videos about some Japanese craftsmen obsessed with details and every single move of a brush or a chisel - think of this lotus 1-2-3 spreadsheet software for DOS. They are of the same quality of craftsmanship.
